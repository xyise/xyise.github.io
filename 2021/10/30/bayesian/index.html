This post is a personal review on the Bayesian Inference.

Prior and Posterial Probabilities

pictorial illustration

A tree diagram would be the simplest way to illustrate the approach.

The left and right sides show prior and posterial probabilities, respectively. Given the prior probabilities P(A)P(A)P(A) and P(B)P(B)P(B) (which is 1−P(A)1-P(A)1−P(A) by definition), the posterial probabilities P(A∣C)P(A|C)P(A∣C), P(B∣C)P(B|C)P(B∣C), P(A∣D)P(A|D)P(A∣D) and P(B∣D)P(B|D)P(B∣D) are the updated probabilities after taking into account the events CCC and DDD (which is the complement of CCC).

The formal derivation is simple

Examples

Covid (or any medical) test

Notation:

  • YYY and NNN are the events of being infected or not infected from Covid, respectively.
  • +++ and −-− indicates the test results, positive and negative, respectively.

Prior probabilities: The test kit should disclose

  • sensitivity (true positive rate): P(+∣Y)P(+|Y)P(+∣Y)
  • specificity (true negative rate): P(−∣N)P(-|N)P(−∣N)

Table with all prior probabilities:

test \ covid Y N
+ P(+∣Y)P( + \vert Y) P(+∣Y): sensitivity, true positive P(+∣N)P(+\vert N)P(+∣N): false positive
- P(−∣Y)P(-\vert Y) P(−∣Y): false negative P(−∣N)P(-\vert N)P(−∣N): specificity, true negative
sum 1 1

Posterior probabilities: When we perform a test, we would like to know

  • P(Y∣+)P(Y|+)P(Y∣+): correct test result
  • P(Y∣−)P(Y|-)P(Y∣−): incorrect test result

Table with all posterial probabilities:

test \ covid Y N sum
+ P(Y∣+)P(Y \vert +)P(Y∣+): correct P(N∣+)P(N \vert +)P(N∣+): incorrect 1
- P(Y∣−)P(Y \vert -)P(Y∣−): incorrect P(N∣−)P(N \vert -)P(N∣−): correct 1

Now, let’s consider the RLF test kit, which has 99.9% sensitivity, or
P(+∣Y)=P(−∣N)=0.999P(+|Y) = P(-|N) = 0.999P(+∣Y)=P(−∣N)=0.999
Then, we have
P(Y∣+)=P(+∣Y)P(Y)P(+∣Y)P(Y)+P(+∣N)P(N)P(Y|+) = \frac{P(+|Y)P(Y)}{P(+|Y)P(Y) + P(+|N)P(N)}P(Y∣+)=P(+∣Y)P(Y)+P(+∣N)P(N)P(+∣Y)P(Y)​
and
P(Y∣−)=P(−∣Y)P(Y)P(−∣Y)P(Y)+P(−∣N)P(N)P(Y|-) = \frac{P(-|Y)P(Y)}{P(-|Y)P(Y) + P(-|N)P(N)}P(Y∣−)=P(−∣Y)P(Y)+P(−∣N)P(N)P(−∣Y)P(Y)​

The prior infection probability P(Y)=1−P(N)P(Y) = 1-P(N)P(Y)=1−P(N) is varied between 0 and 0.001 (0.1%), and an alternative sensitivity assumption 0.990.990.99 (rubbish kit) is also considered:

Suppose that the prevailing infection probabilty is at 1/1000. If the test kit shows a positive result, I have about 50% chance to have a Covid. In comparison, a positive result from a rubbish kit means about 10% chance. In both cases, if negative test result means that there is almost no chance to have a Covid (so, you can go out).
