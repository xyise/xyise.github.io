Note on Mathematical Statistics

  • A dry list of concepts as a personal review note
  • The main references is Wooldridge’s Introductory Econometrics, Appendix C (mostly). Also, wikis and googling.

Among all many concepts, probably the key ones are

  • An estimator is a random variable, thus has its sampling distribution.
  • A (test) statistic is any function of a random sample, so is a random variable itself. Defining the function is entirely up to us.

Law of Large Numbers and Central Limit Theorem: Classical results on sample average

concept description comments

  • Thus, the sample average is a consistent estimator.
  • This is the weak law of large numbers [wiki](https://en.wikipedia.org/wiki/Law_of_large_numbers). If σ2\sigma^2σ2 is assumed to be finite, this law is not only easy to prove but we have a more precise statement, the central limit theorem.

  • Whilst the law of large numbers says the sample average converges to μ\muμ, this class central limit theorem describes the distributional aspect around μ\muμ.

  • One can replace σ\sigmaσ by SnS_nSn​ (sample standard deviation).

Hypothesis Testing and Test Statistic

A hypothesis test is about rejecting or not rejecting a null hypothesis.

concept description comments
null hypothesis The statement being tested.
A typical form is H0:θ=θ0H_0:\theta = \theta_0H0​:θ=θ0​ where θ\thetaθ is the true parameter (or the collection of true parameters or the model itself) based on the population.
  • The null hypothesis is presumed to be true until the data strongly suggest otherwise. Just as a defendent is presumed to be innocent util proven guilty.
  • For example, the hypothetical testing on a null hypothesis H0:μ=0H_0: \mu = 0H0​:μ=0 is not about proving the mean is (this would be a very difficult task if the true value were 0.000001).
alternative hypothesis The statement being tested against the null hypothesis.
A typical form is either one-sided H1:θ<θ0 or H1:θ>θ0H_1: \theta < \theta_0 \ \textrm{or} \ H_1: \theta > \theta_0H1​:θ<θ0​ or H1​:θ>θ0​ or two-sided: H1:θ not equal θ0H_1: \theta \text{ not equal } \theta_0H1​:θ not equal θ0​.
  • To conclude that H0H_0H0​ is false and H1H_1H1​ is true, there should be evidence "beyond reasonable doubt" against H0H_0H0​, which is quantified with respect to the significance level (e.g. 5%).
  • The one-sided alternative hypothesis, say θ<θ0\theta < \theta_0θ<θ0​, is used when it is not of an interest to show the other side of relation θ>θ0\theta > \theta_0θ>θ0​. Showing a defendent is more than innocent is not of the interest in proving guilty.
Type I error & significant level
  • Type I error: Rejecting the null hypothesis when it is in fact true
  • significant level α\alphaα of a test: the probability of a Type 1 error α=P(Type I)=P(Reject H0∣H0)\alpha = P(\text{Type I}) = P(\text{Reject }H_0|H_0)α=P(Type I)=P(Reject H0​∣H0​) or, equivalently α=P(Reject H0;θ=θ0)\alpha = P(\text{Reject }H_0 ; \theta={\color{blue} \theta_0})α=P(Reject H0​;θ=θ0​)
  • Type I error is also known as false negative, which means that H0H_0H0​ is falsely deemed to be positive (rejected).
  • The significant level is set before the test. Common values are 10%, 5% and 1% although there are no clear justifications for them.

Type II error & power
  • Type II error: Failing to reject the null hypothesis when it is in fact false.
  • power of a test π(θ1)\pi({\color{red} \theta_1})π(θ1​) against a relevant and specific alternative θ=θ1\theta = {\color{red} \theta_1}θ=θ1​: π(θ1)=P(Reject H0;θ=θ1)\pi({\color{red} \theta_1}) = P(\text{Reject }H_0 ; \theta={\color{red} \theta_1})π(θ1​)=P(Reject H0​;θ=θ1​) or, equivalently π(θ1)=1−P(Type II;θ=θ1)\pi({\color{red} \theta_1}) = 1 - P(\text{Type II} ; \theta={\color{red} \theta_1})π(θ1​)=1−P(Type II;θ=θ1​)
